Funnel Chart

The Funnel and Pyramid Chart for Power BI acts as two in one visual. It functions in two different visualization modes – Default and 3D. It is highly customizable with an extensive deck of properties for configuring every aspect of the visual. Some of the important features of the visual include Advanced Conditional Formatting and Number Formatting Options on various scenarios present in the chart.

#Default and 3-D Mode

This feature allows you to swap between default view and 3D view mode easily.

#Advanced Conditional Formatting

Alerting Options can be toggled with easily configurable conditional formatting editor

#Number Formatting

This feature allows you to customize values based on various business scenarios

#Funnel / Pyramid View

Visualize data in Funnel view and Pyramid view with the same visual by switching with a click.

#Data Labels

Highly Customisable data label options to give better data insights to business users

#Highly Customizable Color Options

Choose between Power BI Themes and customize colors based on your organizational needs

#Use Cases

Analyze progress made by task groups functioning in an Organization by pictorial representation of data

Handling multiple resources across various timelines can be catered using this visual

Additional information regarding projects can be viewed on Power BI itself, without having to use additional tools
