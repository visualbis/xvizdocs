Gantt Chart

The Gantt Chart for Power BI shows activities (tasks or events) displayed along a given timeline. On the left side of the chart is the Data Grid, which displays lists all the tasks as part of the project or work breakdown the structure to be completed. It is one of the most widely used project planning tools. It helps you visualize your project plan to articulate, communicate and monitor the progress.

#Conditional Formatting

The projects that have missed deadlines can be spotted easily using the advanced conditional formatting option

#Multi-Mode Functionality

This Feature enables you to Switch between Gantt chart & Gantt Resource chart visualization in One Click

#Zoom Range

This feature enables you to range on your zoom to have a dynamic visual in the Gantt Chart

#Status Flags

Status Flag Alerting in addition to conditional formatting to spot the projects that are over running deadlines

#Display Additional Columns

Apart from the project progress displayed in the Gantt Chart, the visual enables you to add more columns to provide information

#Milestones

Milestones can be added to the Gantt Chart to keep track of deliverables

#Date Formats

Extensive Support for the Date formats to provide insights and to track performance in Gantt Chart for Power BI

#Use Cases

Analyze progress made by task groups functioning in an Organization by pictorial representation of data

Handling multiple resources across various timelines can be catered using this visual

Additional information regarding projects can be viewed on Power BI itself, without having to use additional tools
