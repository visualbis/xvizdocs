The Linear Gauge for Power BI can be best used to visualize a KPI. The custom visual enables you to visually compare the actual and target values. 
The color bar shows how much progress has made toward key targets such as month on month or year over year.
It also gives you the ability to see in what range the value of interest falls in the gauge for easy comparison and analysis. 
Configuration options such as orientation, conditional formatting, Number formatting, etc. are available in xViz Linear gauge to customize the appearance of gauges.

### Conditional Formatting
This feature allows the Linear Gauge to be split into bands based on the Target and Maximum Values.
